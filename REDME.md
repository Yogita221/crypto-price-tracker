# Real-Time Crypto Price Tracker 📈

A cryptocurrency price tracking app that simulates real-time price changes and displays detailed market data. Built using React, Redux Toolkit, and Tailwind CSS, this project provides a fast, clean, and extensible frontend suitable for production use or further integration with real APIs.


## 🚀 Setup Instructions

1. **Create a new Vite project**
   ```bash
   npm create vite@latest crypto-price-tracker -- --template react
   cd crypto-price-tracker
   ```

2. **Install required dependencies**
   ```bash
   npm install @reduxjs/toolkit react-redux
   npm install tailwindcss postcss autoprefixer
   npx tailwindcss init -p
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Access the application**
   - Open your browser and navigate to `http://localhost:5173`
   - The app should automatically reload if you make any changes to the source files

## 🧱 Tech Stack + Architecture

### Core Technologies
- **React** - Component-based UI library for building interactive interfaces
- **Redux Toolkit** - State management solution with simplified setup
- **Tailwind CSS** - Utility-first CSS framework for rapid styling
- **Vite** - Modern frontend build tool offering instant server start


## 🎯 Features

- Real-time price updates
- Interactive price charts
- Customizable watchlist
